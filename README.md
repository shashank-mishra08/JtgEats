# JTGeats - Authentic Home Food Delivery

JTGeats is a modern and responsive website for a food delivery service specializing in authentic home-cooked meals. This project is a single-page application (SPA) that provides a seamless user experience for browsing the menu, discovering popular dishes, and contacting the service.

## Features

*   **Responsive Design:** The website is fully responsive and works on all devices, from mobile phones to desktops.
*   **Dynamic Carousel:** A sleek carousel showcases popular food items with smooth transitions and autoplay functionality.
*   **Interactive Menu:** A grid-based menu displays a variety of dishes with details like price, rating, and delivery time.
*   **Video Showcase:** An engaging video section highlights the food and cooking process.
*   **Request a Dish Modal:** A user-friendly modal allows customers to request custom dishes.
*   **Contact Form:** A simple and effective contact form for inquiries and seller applications.
*   **Smooth Scrolling:** Smooth scrolling for anchor links provides a better navigation experience.

## Technologies Used

*   **HTML5:** For the structure and content of the website.
*   **CSS3:** For styling the website, including responsive design and animations.
*   **JavaScript (ES6):** For dynamic functionality, including the carousel, modal, video player, and form submissions.
